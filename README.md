Task-1: Student Performance Data Analysis Using Python

Project Overview
This project focuses on analyzing student exam performance using Python.  
The dataset contains academic, demographic, and study-related information of students.  
The objective is to perform data loading, cleaning, analysis, and visualization using basic Python libraries.

Objectives
- Load and explore the student performance dataset
- Clean the dataset by handling duplicates and missing values
- Perform basic statistical analysis
- Visualize important insights using graphs
- Answer specific analytical questions using Python

Dataset Information
- Dataset Name:`student-mat.csv`
- Description: Student performance in Mathematics
- Attributes Used:
  - `G1`, `G2`, `G3` – Grades for three terms
  - `studytime` – Weekly study time
  - `sex` – Gender of student (M/F)

Technologies Used
- Programming Language:Python 3.x
- Libraries:
  - pandas
  - numpy
  - matplotlib
